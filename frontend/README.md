# frontend

Vue 3 + TypeScript + Vite.

## Requirements

Node 22 (see `.nvmrc`).

## Project setup
```
npm install
```

### Start dev server (hot-reload)
```
npm run dev
```

### Build for production
```
npm run build
```

### Preview the production build
```
npm run preview
```

### Run unit tests
```
npm test
npm test:cov
```

### Lint
```
npm run lint        # eslint
npm run lint:style  # stylelint
```
